This site is a work in progress, currently in Beta. The aim is to have a web client for a physical block-based game that will allow players to choose from options to select their 54 rules to match their 54 blocks.

This is an Angular app with an Express/NodeJS backend.

This repo for this project can be found on github at https://github.com/adam-on-the-internet/fiftyfour-blocks.
